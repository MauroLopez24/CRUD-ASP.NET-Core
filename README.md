CRUD ASP.NET Core
Proyecto CRUD desarrollado con ASP.NET Core y SQL Server.
Tecnologías utilizadas
* C#
* ASP.NET Core
* SQL Server
* HTML / CSS
* Bootstrap
Descripción
Este proyecto implementa un sistema CRUD (Create, Read, Update, Delete) para la gestión de contactos.
Permite:
* Listar contactos
* Crear nuevos contactos
* Editar información
* Eliminar registros
Base de datos
El sistema utiliza SQL Server con una base de datos llamada **DBCRUDCORE**.
Objetivo
Este proyecto forma parte de mi práctica para mejorar mis habilidades en desarrollo backend con .NET y gestión de bases de datos.
